## 2. THE FUNCTIONAL MATRIX

### 2.1 Grammar Prefixes (Operational Commands)

| Code | Meaning | Function |
| :--- | :--- | :--- |
| **o- / 4o-** | The Object | Noun marker; denotes a static substance (e.g., `o-Tar` = Resin). |
| **y- / 8-** | To Action | Verb marker; denotes a command or process (e.g., `y-Tar` = To thicken). |
| **qo-** | Instrumental | Denotes the specific tool or catalyst used (e.g., `qo-sal` = using salt). |
| **chk- / sh-** | Audit | Quality control; verification of temperature or purity. |
